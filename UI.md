🎨 设计方案：「星图纪事」
基于对现有代码库的完整分析，我提出以下方案。核心理念——把"填写表单"变成"探索宇宙"。

美学方向：天文志 · Celestial Cartography
灵感来自古代星图、天体仪、深空观测站。这是一个关于创造世界和故事的工具，界面应该像是在操控一台精密的叙事观测仪。


色彩系统：
背景    #0B0F1A  深空蓝（主背景）
面板    #111827  暗钢蓝（卡片/面板）
边框    #1E293B  铁灰（分隔线）
文字    #E2E8F0  月白（正文）
强调    #D4A853  古金（操作/高亮）
次要    #60A5FA  星蓝（链接/标签）
危险    #EF4444  红移（错误/删除）
成功    #34D399  绿光（完成状态）
各页面改造方案
1. 侧边栏 → 「观测台导航」

半透明毛玻璃效果 backdrop-blur
导航项带微光 hover 动效，active 状态有金色左条
底部设置按钮改为齿轮旋转 hover
2. 首页 → 「星图入口」

背景加一层粒子/星点动画（纯 CSS）
项目卡片改为悬浮玻璃面板，hover 上浮 + 金边
新建按钮改为发光的「+」圆形按钮
3. 人物页 → 「关系星图」

核心改动：用 @xyflow/react（已安装）做力导向图
每个角色是一个发光节点，关系是连线
主角节点更大/更亮，按角色类型着色
点击节点弹出详情面板（侧滑）
保留列表视图作为切换选项
4. 模拟页 → 「时间线仪表盘」

上下分层：顶部是控制台（紧凑），下方是时间线
Tick 按钮组独立为固定操作栏
事件用竖向时间线展示（左线 + 节点），不再是卡片列表
每个事件节点带类型颜色标记
5. 叙事页 → 「叙事织机」

左侧：叙事列表（竖向标签页）
右侧：节拍时间线 + 嵌套场景
节拍用横向流程条展示，可展开看场景
场景对白用剧本格式（角色名高亮）
6. 导出页 → 「档案馆」

简洁：叙事选择 → 格式选择 → 一键生成
导出历史用列表 + 下载图标
技术选型
需求	方案
色彩/主题	CSS 变量 + Tailwind 自定义主题
3D 力导向图	@xyflow/react（已安装）+ 自定义节点
动画	CSS transitions + keyframes（零依赖）
字体	Google Fonts: Noto Serif SC（标题）+ Inter（正文）
图标	Lucide React（轻量 SVG 图标库）
实施顺序
基础层：主题色 + 全局 CSS + 布局骨架
侧边栏：新的导航组件
人物页：力导向图（最大亮点）
模拟页：时间线仪表盘
叙事页：叙事织机
首页 + 导出 + 设置：收尾
